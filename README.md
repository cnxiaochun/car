智能四轮车
============


## 组件

- 铝合金底盘279mm * 183mm * 2mm, 210g
- 电机(370)
- STM32开发板(STM32F103C8T6)
- PS2遥控器
- 18650充电电池 * 3
- 85mm轮胎, 海绵内胆, 轮毂61mm, 轮宽32mm
